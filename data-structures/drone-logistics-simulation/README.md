# Drone Logistics Simulation

This project implements custom data structures and uses them to simulate logistics operations performed by drones.

## Data Structures Implemented

### BetterStack
- Stack using a dynamically resizing array
- Push, Pop, Peek operations
- Generic type support
- Automatic expansion and shrinking

### BetterQueue
- Circular array queue
- O(1) enqueue/dequeue
- Dynamic resizing
- Memory efficient contiguous storage

## Drone Simulations

### Item Request Manager
- Simulates drones retrieving items from storage and delivering them based on request times

### Clean Sword Manager
- Simulates a queue of swords being cleaned and reused based on request timing

## Concepts Demonstrated
- Dynamic arrays
- Circular buffers
- Simulation algorithms
- Event-based systems
- Java generics
